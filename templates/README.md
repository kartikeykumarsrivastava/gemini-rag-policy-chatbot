# Gemini RAG Policy Chatbot

An enterprise-grade Retrieval-Augmented Generation (RAG) chatbot built using:
- Google Gemini
- FAISS Vector DB
- Flask
- PDF-based knowledge base

## Features
- Ask questions on company policy PDF
- Secure RAG architecture
- No data is sent for model training
- Works locally for POC

## How to Run

1. Install dependencies
2. Set GEMINI_API_KEY
3. Run PDF ingestion
4. Start Flask app
